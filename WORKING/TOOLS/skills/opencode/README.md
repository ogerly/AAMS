## Warum dieser Skill?

OpenCode hat ein eigenes Skill-Discovery-System das auf `SKILL.md` Dateien in `~/.config/opencode/skills/` und `.opencode/skills/` basiert.

Dieser Ordner enthält die AAMS-Integration für OpenCode.

---

## Verzeichnis

| Datei | Zweck |
|-------|-------|
| `aams-guard.md` | Guard-Pattern für OpenCode |
| `bootstrap.md` | Bootstrap-Prompt für OpenCode |
| `skill-discovery.md` | Skill-Discovery für OpenCode |
